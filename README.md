# 💰 Tip Calculator

A simple and beginner-friendly web app to calculate the total bill amount including tip.

## 🚀 Features

- Enter bill amount
- Enter tip percentage
- Instantly see the total amount
- Clean and responsive design

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript (vanilla)
